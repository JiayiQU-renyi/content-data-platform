04 / About

How this works.

Architecture

Frontend (ECharts on HTML/CSS/JS)
    ↑ JSON
ADS Layer · 5 business metric tables
    ↑
DWS Layer · subject-area aggregations
    ↑
DWD Layer · cleaned star schema
    ↑
ODS Layer · raw data in Parquet format
    ↑
Raw sources · IMDb + MovieLens + TMDb

Stack

  • storageParquet · Hive-style partitioning
  • computePySpark · Spark SQL
  • olapClickHouse · MergeTree
  • streamingKafka · Flink
  • frontendHTML · CSS · ECharts

Methodology

所有数据通过四层数仓加工：原始数据 → 明细 → 主题聚合 → 业务指标。 采用维度建模（星型模型），事实表按年份做 Hive 风格分区。 本项目纯学习用途，遵循各数据源的非商业使用协议。
